# Contributing to Packet Sniffer

Thank you for your interest in contributing! Please read the guidelines below.

## How to contribute

1. Fork the repository and clone to your local machine.
2. Create a new branch for your feature or fix.
3. Commit your changes with clear messages.
4. Open a pull request describing your changes.

## Code Style

- Write clean, readable C code, following standard conventions.
- Document functions and modules.
- Prefer portable code for multi-platform compatibility.

## Issues & Feature Requests

- Please use the issue templates when opening a new issue.
- Clearly describe bugs, enhancement ideas, or questions.
