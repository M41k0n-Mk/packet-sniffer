# Packet Sniffer Roadmap

## MVP
- Real-time packet capturing
- Basic protocol decoding (Ethernet, IP, TCP, UDP)
- Simple interface for CLI usage

## V1
- Advanced protocol decoding (HTTP, DNS, ARP)
- Filtering system by protocol, address, port
- Export to .pcap and .csv
- Basic GUI

## V2
- Didactic visualization modes
- Pattern detection for security analysis
- Integration with Wireshark
- Documentation and tutorials
