# Packet Samples

This directory contains sample packet capture files for testing and demonstration purposes.

## File Formats

- `.pcap` - Standard packet capture format
- `.pcapng` - Next-generation packet capture format
- `.csv` - Exported packet data

## Usage

These samples can be used to:
- Test packet analysis features
- Demonstrate protocol decoding
- Validate filtering capabilities
- Provide educational examples

## Note

Do not commit sensitive or private network data to this repository.
